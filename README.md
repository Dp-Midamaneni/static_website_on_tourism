# Static Website on Tourism

A simple static tourism website built using **HTML**, **CSS**, and **Bootstrap 4**.  
This project showcases famous tourist places such as the **Taj Mahal** and **Golden Temple**, with a home page, destination list, and detailed views with image carousels.

## Features

- Attractive landing page with a tourism theme
- Favourite places section
- Detailed view pages for tourist locations
- Bootstrap carousel for image gallery display
- Responsive layout for different screen sizes
- Clean and simple UI using custom CSS

## Project Files

- `tourism.html` – Main HTML file
- `tourism_css.css` – CSS file for styling
- External Bootstrap, jQuery, Popper.js, and UI kit scripts are used for layout and interactivity

## Technologies Used

- HTML5
- CSS3
- Bootstrap 4
- JavaScript

## How to Run

1. Clone this repository
2. Make sure `tourism.html` and `tourism_css.css` are in the same folder
3. Open `tourism.html` in any web browser
